# Taiyo Style P.S. Skill

## Overview

**成約率を最後の3%押し上げる**太陽スタイルの追伸（P.S.）生成スキル。
読者の73%が追伸を読むというデータに基づき、最終決断を促す追伸パターンを作成。

## When to Use

```
「太陽スタイルで追伸を書いて」
「セールスレターの追伸を作って」
「/taiyo-style-ps」
「メルマガの追伸を最適化して」
```

## 追伸7つの黄金パターン

### パターン1: 緊急性強調型
```
【目的】今すぐ行動させる

【テンプレート】
P.S.
締切は◯月◯日23:59です。
残り◯時間で、この機会は完全に消えます。

1日遅れるごとに、あなたは◯◯を失い続けます。
今この瞬間が、最も安く、最も有利な条件で
手に入れられる最後のチャンスです。

【キラーワード】
「残り◯時間」「完全に消える」「最後のチャンス」「今この瞬間」
```

### パターン2: 損失回避型
```
【目的】行動しない痛みを最大化

【テンプレート】
P.S.
正直に言います。
この手紙を読んで何もしない人が、
おそらく90%以上でしょう。

でも、それはあなたの自由です。
ただ、1年後に「あの時、始めていれば...」と
後悔しないでください。

あの時の自分を恨まないでください。
今日の決断が、1年後のあなたを作ります。

【キラーワード】
「後悔」「1年後」「あの時」「恨まない」
```

### パターン3: 希望提示型
```
【目的】成功した未来を想像させる

【テンプレート】
P.S.
3ヶ月後のあなたを想像してみてください。

毎朝、目覚まし時計なしで起きる。
好きな時間に好きなカフェで仕事をする。
月末の支払いに怯えない生活。

その未来は、今日の「YES」から始まります。
あなたの新しい人生が、今、始まろうとしています。

【キラーワード】
「想像してみて」「◯ヶ月後」「新しい人生」「始まる」
```

### パターン4: 保証強調型
```
【目的】リスクを完全に取り除く

【テンプレート】
P.S.
まだ迷っていますか？

大丈夫です。
30日間の完全返金保証があります。

万が一、内容に満足できなければ、
理由を問わず全額をお返しします。
メール1本で手続き完了。面倒なやりとりは一切なし。

つまり、あなたにリスクは一切ありません。
リスクはすべて私が負います。

まずは試してみてください。

【キラーワード】
「完全返金」「リスクゼロ」「試してみて」「私が負います」
```

### パターン5: 特典追加型
```
【目的】最後のプッシュで価値を追加

【テンプレート】
P.S.
実は、まだお伝えしていないことがあります。

今日中にお申し込みいただいた方には、
特別に「◯◯◯」をプレゼントします。

これは通常◯◯万円で販売しているもので、
今回だけの完全限定特典です。

明日にはこの特典は消えます。
後から「欲しい」と言われても、お渡しできません。

【キラーワード】
「まだお伝えしていない」「特別に」「完全限定」「明日には消える」
```

### パターン6: 社会的証明型
```
【目的】他の人の行動で安心させる

【テンプレート】
P.S.
この手紙を公開してから、
すでに◯◯名の方がお申し込みくださっています。

「もっと早く知りたかった」
「人生が変わりました」
という声を、毎日のようにいただいています。

次はあなたの番です。

【キラーワード】
「すでに◯名」「毎日のように」「次はあなたの番」
```

### パターン7: 個人的メッセージ型
```
【目的】親密さと信頼で心を動かす

【テンプレート】
P.S.
最後に、個人的なことを言わせてください。

私がこのプログラムを作った理由は、
かつての自分のような人を一人でも減らしたいからです。

あの時の私は、本当に苦しかった。
誰にも相談できず、一人で抱え込んでいました。

だから、あなたには同じ思いをしてほしくない。
私はあなたの味方です。一緒に乗り越えましょう。

【キラーワード】
「個人的なこと」「かつての自分」「あなたの味方」「一緒に」
```

## 追伸の黄金法則

### 法則1: 本文を読まなくても理解できる
```
追伸だけ読んでも、
・何のオファーか
・何がお得か
・いつまでか
がわかるように書く
```

### 法則2: 最重要情報を再掲
```
本文で伝えた中で最も重要な：
・締切
・限定性
・ベネフィット
・保証
のいずれかを再掲する
```

### 法則3: 感情で終わる
```
❌ 「お申し込みはこちらから」（事務的）

✅ 「あなたの新しい人生が、今、始まります」（感情的）
```

### 法則4: P.S.は複数使える
```
P.S.   → 緊急性
P.P.S. → 保証
P.P.P.S. → 個人的メッセージ

※3つまでが効果的。4つ以上は逆効果
```

### 法則5: リンクは必ず入れる
```
デジタルの追伸には、
必ずCTAリンクを入れる。

「今すぐ確認する → [リンク]」
```

## 太陽パターン適用

### 語尾バランス（追伸版）
```
断定形: 50%
行動喚起型: 30%
疑問形: 10%
その他: 10%
```

### キラーワード配置
```
冒頭: 緊急性ワード（「残り」「最後」「今日」）
中盤: 損失ワード（「失う」「後悔」「消える」）
終盤: 希望ワード（「始まる」「変わる」「手に入る」）
```

### 心理トリガー
```
必須: 緊急性・希少性トリガー
推奨: 損失回避トリガー
補助: 希望・快楽トリガー
```
