# 太陽スタイル・コピーライティングスキル

日給5000万円、82,187,718円を生み出した太陽スタイルの全技術を適用。

## When to Use This Skill

以下の場合にこのスキルを使用：
- 「太陽スタイルで書いて」「セールスレターを作成」
- 「ステップメールを作成」「メルマガを書いて」
- 「売れるコピーを書いて」
- 「高成約率のコピーが欲しい」

## 太陽スタイルの3つの柱

### 1. 対話型コピー（疑問形33%）
読者に考えさせ、自分で答えを出させる。セールスされた感覚ゼロ。

### 2. 感情のジェットコースター
恐怖→希望→恐怖→希望を20回以上繰り返し、読者の脳を揺さぶる。

### 3. 数値とストーリーの融合
「82,187,718円」の具体性と「5000万盗られた」の人間臭さを両立。

## 商品別カスタマイズ

### 高額商品（30万円以上）
1. 権威の4層構築（実績→経験→知識→人格）
2. 説明形語尾（のです/なのです）を30%使用
3. 90日返金保証＋分割払い提示

### 中額商品（10-30万円）
1. 具体的成功事例を3つ以上
2. 費用対効果の数値化（投資回収率10,000%など）
3. 対比ストーリー（従来型 vs あなたの商品）

### 低額商品（10万円以下）
1. 超短文52%＋感嘆文12%でテンポ加速
2. 今すぐ使える即効性を強調
3. 箇条書き頻度UP

## 業界別カスタマイズ

### 稼ぐ系（副業、投資、転売）
- 数値の階段でアンカリング
- 恐怖喚起（95%が失敗、AIに仕事を奪われる）
- 時間短縮を強調

### 非稼ぐ系（恋愛、美容、健康）
- 感覚語を多用（視覚32回、触覚12回、聴覚8回）
- 自己実現の希望
- 体言止めを30%以上

### BtoB（企業向けサービス）
- 断定形70%で専門性と確実性
- ROI・費用対効果を徹底的に数値化
- 権威の証明（固有名詞42回）

## ステップメール構成

### 第1通（信用の教育）
- 自己紹介＋実績の数値化
- 自己開示（弱さ→強さ）
- 価値提供（有益情報の無料提供）

### 第2通（問題点の教育）
- 95%が失敗する理由
- 恐怖の数値化（損失の具体化）
- 競合増加の警告

### 第3通（手段の教育）
- 解決策の具体的提示（3ステップなど）
- 従来型 vs 新しい方法の対比
- 小さな成功体験の促進

### 第4通（投資の教育）
- 無料派 vs 投資派の明確な対比
- 費用対効果の異常性
- 時間という資源の価値

### 第5通（行動の教育）
- 道A（行動）vs 道B（後悔）の最終対比
- 時間的希少性（今日23:59まで）
- 強い命令形（行動してください）

## 失敗パターンTOP5

1. **疑問形が少なすぎる** → 33%に増やす
2. **数値が抽象的** → 端数まで記載
3. **長文だらけ** → 短文42%、1文1行87%
4. **感情の起伏がない** → 恐怖→希望を5回以上
5. **証拠がない** → 1主張につき1証拠
