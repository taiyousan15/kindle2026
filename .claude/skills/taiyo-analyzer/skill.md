# 太陽 Analyzer Skill

## Overview

セールスレター、LP、ステップメールなどのコンテンツを**太陽スタイル176パターン**に基づいて分析し、0-100点でスコアリング。具体的な改善提案を自動生成する。

## 使用方法

```
「このセールスレターを太陽スタイルで分析して」
「LP分析して改善点を教えて」
「この文章の太陽スコアは？」
```

## 分析フレームワーク

### 6次元スコアリングシステム

```
┌────────────────────────────────────────────────────┐
│              太陽 Score (0-100)                     │
├────────────────────────────────────────────────────┤
│  1. 語尾パターン分析      (20点)                    │
│  2. キラーワード配置      (20点)                    │
│  3. 心理トリガー使用      (20点)                    │
│  4. 文章リズム分析        (15点)                    │
│  5. ストーリー構造        (15点)                    │
│  6. 全体バランス          (10点)                    │
├────────────────────────────────────────────────────┤
│  Total: 100点満点                                   │
└────────────────────────────────────────────────────┘
```

### 12種類の語尾パターン

#### 1. 疑問形・問いかけ型 (理想: 25-30%)
```

**心理効果**: 読者の思考を刺激、自己対話を促進

#### 2. 断定形・言い切り型 (理想: 55-60%)

**心理効果**: 共感を生み、一体感を創出

#### 4. 共感・同意誘導型 (理想: 5-10%)

**心理効果**: 心理的距離を縮める

#### 8. 比較・対比型 (理想: 2-4%)

#### 9. 引用・証言型 (理想: 2-4%)

#### 11. 省略・余韻型 (理想: 2-3%)

#### 12. カジュアル・親近感型 (理想: 1-3%)
```

## 3. 心理トリガー分析 (20点)

### 5層構造

#### 第1層: 損失回避・恐怖トリガー
```
・このまま何もしなければ...
・気づいた時にはもう遅い
・取り返しのつかない失敗
・人生が詰む前に
・手遅れになる前に
```
**配置**: オープニング〜ボディ前半

#### 第2層: 希望・快楽追求トリガー
```
・人生が180度変わる
・夢のような生活
・理想の未来
・自由な人生
・経済的自由
```
**配置**: ボディ中盤

#### 第3層: 社会的証明トリガー
```
・すでに◯名が実践
・実績者の声
・メディア掲載
・専門家推薦
・数字による証明
```
**配置**: ボディ全体に散りばめる

#### 第4層: 希少性・緊急性トリガー
```
・今だけの特別価格
・限定◯名様
・残りわずか
・二度とない機会
・締切迫る
```
**配置**: クロージング直前〜クロージング

#### 第5層: 権威・専門性トリガー
```
・◯年の実績
・◯人を指導
・業界のパイオニア
・専門的知見
・独自メソッド
```
**配置**: ボディ〜クロージング

### セクション別理想配置
| セクション | 主要トリガー | 補助トリガー |
|-----------|------------|-------------|
| オープニング | 損失回避・恐怖 | 希望・快楽 |
| ボディ前半 | 損失回避・恐怖 | 社会的証明 |
| ボディ中盤 | 希望・快楽 | 権威・専門性 |
| ボディ後半 | 社会的証明 | 希少性・緊急性 |
| クロージング | 希少性・緊急性 | 損失回避 |
| 追伸 | 緊急性 | 恐怖 |

### 心理トリガー10の黄金法則
1. **恐怖→希望サイクル**: 最低5回以上繰り返す
2. **三層脳アプローチ**: 爬虫類脳・哺乳類脳・人間脳に同時刺激
3. **感情ジェットコースター**: 振幅を最大化
4. **具体的数字**: 抽象より具体
5. **ストーリー埋め込み**: トリガーをストーリーに包む
6. **五感刺激**: 視覚・聴覚・触覚を活用
7. **未来投影**: 読者に成功した姿を想像させる
8. **損失フレーミング**: 得る喜びより失う恐怖
9. **社会的証明の積み重ね**: 複数の証拠を重ねる
10. **緊急性の段階的強化**: 徐々に圧を高める

## 5. ストーリー構造分析 (15点)

### 太陽流ストーリー構造

#### 7幕構成
1. **どん底の始まり** - 最悪の状態からスタート
2. **転機との出会い** - 救いの兆し
3. **試行錯誤** - 失敗と学び
4. **ブレイクスルー** - 突破口発見
5. **成功への道** - 実践と結果
6. **現在の姿** - 変化後の状態
7. **読者への橋渡し** - 同じ道を歩む招待

#### 感情曲線の理想形
```
感情レベル
    ^
100 |        ＿＿＿     ★成功
    |       /    \
 50 |    __/      \__
    |   /            \
  0 |--/              \----
    ★どん底           ★CTA
    +---------------------> 時間
```

### チェックポイント
- [ ] 冒頭でインパクトのある「どん底」描写がある
- [ ] 転機が明確に示されている
- [ ] 失敗→学び→成功のサイクルがある
- [ ] 具体的な数字・成果が含まれている
- [ ] 読者が自分事として感じられる
- [ ] クロージングへの自然な流れがある

### スコアリング基準
| 評価 | 条件 | 点数 |
|-----|------|-----|
| 優秀 | 7幕すべて含む、感情曲線最適 | 14-15点 |
| 良好 | 5-6幕含む、感情曲線良好 | 11-13点 |
| 普通 | 3-4幕含む | 8-10点 |
| 要改善 | 2幕以下 | 5-7点 |
| 不足 | ストーリー性なし | 0-4点 |

## API仕様（他スキルからの呼び出し用）

### 関数シグネチャ
```
taiyo_analyze(content: string, options?: AnalysisOptions): AnalysisResult

interface AnalysisOptions {
  contentType: 'sales-letter' | 'lp' | 'step-mail' | 'email';
  detailLevel: 'summary' | 'standard' | 'detailed';
  includeRewrite: boolean;
}

interface AnalysisResult {
  totalScore: number;
  rank: string;
  scores: {
    endingPattern: number;
    killerWords: number;
    psychologicalTriggers: number;
    rhythm: number;
    storyStructure: number;
    overallBalance: number;
  };
  strengths: string[];
  improvements: Improvement[];
  rewriteSuggestions?: string[];
}
```

### 使用例
```
# sales-letterスキルから呼び出し
result = taiyo_analyze(draft_content, {
  contentType: 'sales-letter',
  detailLevel: 'detailed',
  includeRewrite: true
})

if result.totalScore < 80:
  apply_improvements(result.improvements)
```
