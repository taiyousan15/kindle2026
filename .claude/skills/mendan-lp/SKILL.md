# 面談LP作成スキル

面談申込率50%を実現する4つの型から最適なLPを作成するスキル。

## When to Use This Skill

以下の場合にこのスキルを使用：
- 「面談LPを作って」「個別相談LPを作成」
- 「高額商品のLPを作りたい」
- 「申込率の高いLPが欲しい」
- 「ストーリー型/タイムライン型/Q&A型/神話崩し型でLPを」

## 4つの型の特徴

| 型 | 構成 | 強み | 最適なケース |
|---|------|------|------------|
| **ストーリー型** | 15パート | 感情移入・記憶定着最高 | コーチング、美容、健康 |
| **タイムライン型** | 10パート | 時間価値認識・論理的納得 | 投資、副業、スキルアップ |
| **Q&A型** | 13パート | 論理的信頼・低キャンセル率 | 医療、金融、法律、高額商品 |
| **神話崩し型** | 11パート | 衝撃的覚醒・最強の印象 | 革新的手法、業界改革 |

## 型選択フローチャート

### Step1: ターゲットの思考タイプ
- 感情重視 → ストーリー型 or 神話崩し型
- 論理重視 → タイムライン型 or Q&A型

### Step2: 商品・サービスの特性
- 体験・変革系 → ストーリー型
- 成長・投資系 → タイムライン型
- 複雑・専門系 → Q&A型
- 革新・差別化系 → 神話崩し型

### Step3: 市場環境
- 競合が多い → 神話崩し型
- 時間制約あり → タイムライン型
- 不安が多い → Q&A型
- 感動を求める → ストーリー型

## 科学的基盤

## LP作成フロー

1. **ターゲット情報の収集**
   - 年齢・性別・職業
   - 主な悩み・欲求
   - 思考タイプ（感情/論理）

2. **型の選択**
   - 上記フローチャートで最適な型を選択

3. **各パートの作成**
   - 選択した型のテンプレートに沿って作成

4. **心理学的テクニックの適用**
   - 各型専用のテクニックを適用
