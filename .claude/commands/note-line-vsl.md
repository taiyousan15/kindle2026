# note-LINE-VSLファネルスキル

note記事からLINE登録→VSL動画へ誘導するフルファネルを構築するスキルです。
UTAGEでLINEステップ配信、OmniHuman1でAIアバターVSLを生成します。

## 使用方法

```
/note-line-vsl
```

---

## 全体フロー

```
[note記事] → [LINE登録(CTA)] → [LINEステップ(UTAGE)] → [VSL動画(OmniHuman1)] → [商品購入]
```

---

## Step 1: ヒアリング

以下の情報を確認してください：

1. **note記事のテーマ**: 何についての記事？
2. **ターゲット読者**: どんな人が読む？
3. **記事タイプ**: 無料記事 or 有料記事？
4. **バックエンド商品**: 最終的に何を売る？
5. **特典内容**: LINE登録でもらえる特典は？
6. **VSLの長さ**: 7分 or 15分？

## Step 4: QRコード・リンク設定

### UTAGEでQRコード生成

```
【手順】
1. UTAGEにログイン
2. LINE設定 → QRコード生成
3. 画像をダウンロード（PNG）
4. サイズ: 200x200px以上

【noteへの挿入方法】
1. note編集画面
2. 画像を挿入
3. 中央揃え
4. QRコードの下にリンクテキストも記載
   （スマホで見ている人用）
```

### リンクの書き方

```markdown
▼ LINE登録はこちら ▼
https://lin.ee/XXXXXXX

↑タップして登録できます
または下のQRコードを読み取ってください

[QRコード画像]
```

## Step 5: 特典コンテンツ作成

### X/YouTube リサーチ → 特典化

```
【リサーチ手順】

1. Xで関連キーワード検索
   - "[テーマ] コツ"
   - "[テーマ] 方法"
   - バズっている投稿をチェック

2. YouTubeで関連動画検索（英語）
   - "[テーマ] tutorial"
   - "[テーマ] tips"
   - 視聴回数10万回以上を優先

3. 人気コンテンツ10本をリストアップ
   - ポイントを抽出
   - 共通点を分析

4. 日本語にローカライズ

5. Notionで特典ページ作成
   - わかりやすく構成
   - 図解を追加
```

### タグ設定

```
【タグ名】
- note_reader: note経由の登録者
- note_free: 無料記事から登録
- note_paid: 有料記事から登録
- watched_vsl: VSL視聴者
- clicked_offer: オファークリック者
- purchased: 購入者

【分岐例】
- note_paid → 購入者限定特典を追加配信
- watched_vsl → 購入促進メッセージ
```

## note記事タイプ別の最適化

### 無料記事（集客用）

```
目的: LINE登録を最大化

【ポイント】
- 価値を出し惜しみしない
- 記事の「続き」で誘導
- CTAは最低3箇所
- シェアされやすい内容

【LINE誘導の訴求】
「この記事の内容をさらに深掘りした
○○をLINEで配布中」
```

### 有料記事

```
目的: 購入者をLINEに誘導→アップセル

【ポイント】
- 購入者限定特典として訴求
- 「購入者だけに」という限定感
- アップセル商品への導線

【LINE誘導の訴求】
「購入者限定！この記事の実践版
○○テンプレートをLINEでプレゼント」
```

### シリーズ記事

```
目的: シリーズ全体でファン化→LINE

【ポイント】
- シリーズ全体の特典をLINEで
- 更新通知をLINEで
- コミュニティ感を演出

【LINE誘導の訴求】
「このシリーズの更新通知と
限定コンテンツをLINEでお届け」
```
