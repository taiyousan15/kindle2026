# Kindle-LINE-VSLファネルスキル

Kindle本からLINE登録→VSL動画へ誘導するフルファネルを構築するスキルです。
UTAGEでLINEステップ配信、OmniHuman1でAIアバターVSLを生成します。

## 使用方法

```
/kindle-line-vsl
```

---

## 全体フロー

```
[Kindle本] → [LINE登録(QRコード)] → [LINEステップ(UTAGE)] → [VSL動画(OmniHuman1)] → [商品購入]
```

---

## Step 1: ヒアリング

以下の情報を確認してください：

1. **Kindle本のテーマ**: 何についての本？
2. **ターゲット読者**: どんな人が読む？
3. **バックエンド商品**: 最終的に何を売る？
4. **特典内容**: LINE登録でもらえる特典は？
5. **VSLの長さ**: 7分 or 15分？
6. **アバター設定**: 性別、年代、雰囲気

## Step 2: Kindle本へのLINE誘導設置

### 設置場所（推奨）

```
【必須設置ポイント】
1. 本の冒頭（はじめに/プロローグの後）
2. 各章の冒頭（Chapter 1, 2, 3...）
3. 本の最後（あとがき/おわりに）

【推奨設置数】
- 10章の本: 12-15箇所
- 5章の本: 7-10箇所
```

### LINE誘導テンプレート（Kindle用）

```markdown
━━━━━━━━━━━━━━━━━━━━━━━━━━
【読者限定プレゼント】

この本の内容を実践するための
「○○完全ガイド」を無料でプレゼント中！

▼ 今すぐ受け取る ▼
https://lin.ee/XXXXXXX

※スマホでQRコードを読み取るか、
　上記URLをタップしてください
━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### QRコード生成

```
【UTAGE公式手順】
1. UTAGEにログイン
2. LINE連携設定 → QRコード生成
3. 画像をダウンロード（PNG形式）
4. サイズ: 300x300px推奨

【Kindle埋め込み】
- epub形式で画像挿入
- alt属性: "LINE登録QRコード"
- 前後に説明テキストを配置
```

### 章ごとの特典訴求バリエーション

```
【Chapter 1 冒頭】
「この章の内容を深掘りした動画解説を
LINE登録者限定で配信中！」

【Chapter 3 冒頭】
「[章のテーマ]の実践テンプレートを
LINE限定で無料配布中！」

【Chapter 5 冒頭】
「実際に[成果]を出した事例集を
LINE登録で今すぐ受け取れます」

【最終章 / あとがき】
「本書の内容を1枚にまとめた
チートシートをプレゼント中！」
```

## Step 3: 特典コンテンツ作成

### X/YouTube リサーチ → 特典化

```
【リサーチ手順】

1. YouTubeで関連キーワード検索（英語）
   - "[テーマ] tutorial"
   - "[テーマ] how to"
   - "[テーマ] tips"

2. 人気動画10本をリストアップ
   - 視聴回数10万回以上を優先
   - 最新（1年以内）を優先

3. 各動画からキーポイントを抽出
   - タイトルの訴求ポイント
   - サムネイルの要素
   - コメント欄の質問/反応

4. 日本語に翻訳・ローカライズ

5. 特典コンテンツ化
   - PDFガイド
   - チェックリスト
   - テンプレート
   - 動画解説
```

### 特典フォーマット（推奨）

```
【特典1: メイン特典】
- PDF: 20-30ページ
- 内容: 本の内容を深掘り

【特典2: 実践ツール】
- チェックリスト or テンプレート
- そのまま使える実用的なもの

【特典3: 限定動画】
- 7分程度の解説動画
- 本に書けなかった内容

【ボーナス: VIP特典】
- 購入者限定のさらなる特典
- 緊急性を出す
```

### Notion配信設定

```
【Notion特典ページ構成】

1. ヘッダー
   - 「○○限定特典ページ」
   - アイキャッチ画像

2. 特典一覧
   - 特典1: [ダウンロードリンク]
   - 特典2: [ダウンロードリンク]
   - 特典3: [動画埋め込み]

3. 次のステップ
   - VSL動画への誘導
   - 「さらに詳しく知りたい方は↓」

4. フッター
   - サポート連絡先
   - SNSリンク

【アクセス制限】
- ページをプライベート設定
- 共有リンクを生成
- UTAGEから自動配信
```

### UTAGEステップ配信設定

```
【配信時間設定】
- Day 0: 登録直後（即時）
- Day 1: 翌日 20:00
- Day 2: 翌々日 20:00
- Day 3: 3日後 20:00（VSL送付）
- Day 4-7: 状況に応じて

【タグ設定】
- kindle_reader: Kindle経由の登録者
- watched_vsl: VSL視聴者
- clicked_offer: オファーページクリック者
- purchased: 購入者

【分岐設定】
- VSL視聴 → 購入促進シナリオ
- 未視聴 → リマインドシナリオ
```

### 台本作成のコツ

```
【話し言葉で書く】
- 「です・ます」調
- 1文を短く（30文字以内）
- 専門用語を避ける

【感情を入れる】
- 共感「私も同じでした」
- 期待「想像してみてください」
- 緊急性「今だけ」「限定」

【1分あたり約150-180字】
- 7分台本 = 約1,000-1,200字
- 15分台本 = 約2,200-2,700字
```

## Step 6: OmniHuman1でAIアバター動画生成

### OmniHuman1 概要

```
公式サイト: https://www.omnihuman1.org/ja

【特徴】
- AIアバターによるリップシンク動画生成
- 音声からリアルな口の動きを生成
- 高品質な表情アニメーション
```

### 動画制作ワークフロー

```
【Step 1: 音声収録】
━━━━━━━━━━━━━━━━━━
1. 台本を読み上げて録音
   - 静かな環境
   - クリアな発声
   - 適度なペース

2. 音声編集
   - ノイズ除去
   - 音量調整
   - 無音カット

3. 形式: MP3 or WAV
   - 44.1kHz, 16bit推奨
━━━━━━━━━━━━━━━━━━

【Step 2: アバター選択/作成】
━━━━━━━━━━━━━━━━━━
1. OmniHuman1にアクセス
2. アバターを選択
   - 性別
   - 年代
   - 雰囲気（ビジネス/カジュアル）

3. または自分の写真からアバター生成
━━━━━━━━━━━━━━━━━━

【Step 3: リップシンク生成】
━━━━━━━━━━━━━━━━━━
1. 音声ファイルをアップロード
2. アバターを選択
3. 生成開始
4. 完成動画をダウンロード
━━━━━━━━━━━━━━━━━━

【Step 4: 動画編集（後処理）】
━━━━━━━━━━━━━━━━━━
1. 必要に応じてカット編集
2. テロップ追加（重要ポイント）
3. BGM追加（控えめに）
4. オープニング/エンディング追加
5. CTA画面追加

ツール: Canva動画 or CapCut
━━━━━━━━━━━━━━━━━━
```

## トラブルシューティング

### よくある問題と解決策

```
【QRコードが読み取れない】
- サイズを大きくする（300px以上）
- コントラストを上げる
- 周囲に余白を確保

【LINE登録されない】
- URLが正しいか確認
- UTAGEの連携設定を確認
- テスト登録を実行

【動画が再生されない】
- プライバシー設定を確認
- URLの共有設定を確認
- ファイルサイズを確認

【ステップ配信されない】
- シナリオの有効化を確認
- 配信時間設定を確認
- タグ条件を確認
```
